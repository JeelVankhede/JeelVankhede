# Who Am I?

I write the engineering contracts your AI coding agents start from.

Lead Full Stack Engineer who can cover anything challenged (Mobile, Frontend and Backend) · 9+ years in Industry · Ahmedabad, India

---

Every new project, the same problem: reconfigure Cursor rules from scratch, set up Claude Code again, hope Copilot picks up on your patterns. Each agent works in isolation. By the time all five behave the way you want, you have spent more time on configuration than on code.

**fare** and **bare** fix that. One command generates the full engineering contract — architecture decisions, coding standards, lifecycle, testing practices, review process; and renders it natively into every agent you use. The agents start from your rules. You start from the code.

### [fare](https://jeelvankhede.github.io/frontend-ai-starter-recipes/) — frontend-ai-starter-recipes

AI coding agent rules, skills, and workflows for frontend projects.

Generates 71 files covering architecture, component standards, state management, testing, and review process. Tuned to your UI library and meta-framework (React, Vue, Svelte, Angular). Renders natively into every adapter you use.

```bash
npx frontend-ai-starter-recipes
```

Adapters: Cursor · Claude Code · GitHub Copilot · Windsurf · Antigravity  
Frameworks: React · Next.js · Vue · Nuxt · Svelte · SvelteKit · Angular
